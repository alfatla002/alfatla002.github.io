AI-Assisted JUnit Test Case Schema Generator (IN PROGRESS)

Overview

Research project exploring how Large Language Models (LLMs) can classify GitHub issues and generate structured JUnit test case templates based on user stories. This work supports automating testing workflows in large open-source repositories.

Goals

  • Extract labeled user stories and issues from repositories
  • Use LLMs to classify issue type (bug, enhancement, feature request)
  • Automatically generate JUnit test case schema suggestions
  • Improve developer productivity and testing consistency

Research Context

  • Study of GitHub and Apache repositories where issues lack associated tests
  • Use of issue metadata, comments, and tags for classification
  • Evaluation of LLM performance against human-written examples

Technologies Used

  • Python
  • OpenAI LLMs
  • JUnit (schema modeling)
  • GitHub API
