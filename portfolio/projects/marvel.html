Marvel API Microservice

Overview

A RESTful backend that fetches Marvel character and comic data from the official Marvel API, processes it, and stores records in Supabase for fast reuse. Includes MD5-based signature authentication and rate-limit management.

Technologies Used

  • Node.js (Express)
  • Supabase (Database + Storage)
  • Marvel Developer API
  • MD5 signing algorithm

My Contributions

  • Built RESTful API routes for character & comic ingestion
  • Implemented Marvel MD5 hashing for API authentication
  • Created database schema for caching API responses
  • Added filtering endpoints for searching heroes
  • Optimized ingestion to reduce rate-limit usage
