DevOps, Docker, and Deployment Labs

Overview

A series of hands-on labs completed across multiple courses involving Docker, Node.js deployments, PM2 process management, Supabase self-hosted stack, Nginx static hosting, and environment configuration.

Technologies Used

  • Docker & Docker Compose
  • Nginx static server
  • Supabase self-hosted stack (Kong, Studio, Realtime, Postgres)
  • PM2 (Node.js process manager)
  • Environment configuration (.env, JWT, keys)
  • Ubuntu server hosting & SSH

Key Lab Work

  • Hosted static websites inside Dockerized Nginx
  • Rebuilt Supabase dev stack via Docker Compose
  • Configured and deployed Next.js with PM2
  • Debugged container mount issues and network overlays
  • Managed environment variables securely for cloud deployments
