Remote Rendering System (Lockheed Martin)

Overview

A one-year engineering project sponsored by Lockheed Martin. We developed a distributed split-rendering system that allows low-end devices to display high-polygon 3D models by offloading rendering to remote Unity/ArenaXR instances running inside Docker containers.

Technologies Used

  • Unity Engine
  • ArenaXR
  • Docker (multi-container system)
  • Python automation scripts
  • Google OAuth authentication
  • Cloud rendering over LAN/VPN

My Contributions

  • Integrated Unity client with ArenaXR remote-render pipeline
  • Developed Python scripts to start, stop, and manage render sessions
  • Implemented Google OAuth login system for secure access
  • Captured and analyzed performance metrics (FPS, latency, network load)
  • Built Docker images for hosting remote rendering nodes
  • Documented system architecture and hand-off for Lockheed Martin team
