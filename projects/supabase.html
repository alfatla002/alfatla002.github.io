Supabase Image Gallery & File Browser

Overview

A complete file browser built on Supabase Storage with secure backend routing, multi-file upload support, bucket creation UI, thumbnail preview grid, and drag-and-drop functionality — deployed with Cloudflare + Nginx Proxy Manager.

Features

  • 4×3 thumbnail gallery with real previews
  • Multi-file upload + drag and drop
  • Batch delete and batch download actions
  • Bucket creation UI
  • File browser with checkboxes
  • Backend service-role API for secure Storage uploads
  • Custom confirm dialog for drag-and-drop uploads

Technologies Used

  • Supabase Storage & Auth
  • Node.js backend (service role key)
  • Vite + JavaScript
  • Cloudflare DNS + Tunnels
  • Nginx Proxy Manager deployment
